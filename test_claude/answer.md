## 3プラン三段構え

### ★ プラン1（冒険型・本命）: 姫路セントラルパーク

> 「姫路城が雨で行けない → 代わりに姫路でサファリ」

- **アクセス**: 東灘区から車約1時間
- **内容**: 雨のドライブスルーサファリ + 遊園地エリア屋内アトラクション
- **なぜ推し?**:
  - 雨のサファリ = 逆転の発想で非日常度MAX
  - 車内から動物を至近距離で見る体験は晴れの日より印象的
  - 息子の生き物好きに直球で刺さる
  - 娘もSNS映え写真が撮れる
  - 遊園地の屋内アトラクションもあり滞在時間が長く取れる
  - 帰りに姫路駅前でお母さんの誕生日ディナー
- **推し**: Tulip 1位、Dean 本命、Nola 賛同

### ★ プラン2（近場充実型）: 神戸どうぶつ王国 + atoa + ポーアイディナー

> ポートアイランド完結、移動ストレスゼロの欲張りコース

- **アクセス**: 東灘区から車15〜20分
- **内容**:
  - 午前: 神戸どうぶつ王国（ハシビロコウ、カピバラ等レア動物）
  - 午後: atoa（水族館×デジタルアート、SNS映え）
  - 夜: ポーアイのホテルレストランで母の誕生日ディナー
- **なぜ推し?**:
  - 天候がどれだけ荒れても安心の確実プラン
  - 移動ゼロでストレスフリー
  - 母の誕生日ディナーまでセットにできる
- **注意**: atoaが「水族館NG」に引っかかるかKenさんに確認を
- **推し**: 全員支持の安全策

### ★ プラン3（カルチャー型・チケット次第）: 宝塚歌劇 + 手塚治虫記念館

> 家族を2チームに分けて、それぞれの「最高」を体験

- **アクセス**: 東灘区から車30分
- **内容**:
  - 母＋娘: 宝塚歌劇観劇（誕生日プレゼントとして最高の非日常）
  - 父＋息子: 手塚治虫記念館（アトム、ブラックジャック等）
  - 夜: 花のみち周辺で合流ディナー「今日どうだった？」と語り合う
- **なぜ推し?**:
  - 母の誕生日祝いとしては最も感動的
  - 息子問題を手塚治虫記念館で解決
  - 夜の合流ディナーで家族の会話が弾む
- **注意**: 4/4金曜なら当日券の可能性あり → 今すぐ宝塚公式サイトで確認を
- **推し**: Dean 対抗、Nola 補足提案
